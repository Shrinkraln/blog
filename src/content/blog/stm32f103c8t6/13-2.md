1. 1. 对于中断触发才运行的代码加上低功耗模式
   1. 睡眠模式可以收到外设的中断
   2. 停机模式只能收到外部中断
   3. 待机模式只能收到四个指定事件退出
2. 停机和待机模式需要开启PWR时钟控制，然后进入指定模式
3. 停止模式要在复位时恢复systeminit()恢复主频，不用__WFE()
4. **待机模式没有区分WFT和WFE**

```c
void PWR_DeInit(void);
void PWR_BackupAccessCmd(FunctionalState NewState);

void PWR_PVDCmd(FunctionalState NewState);
void PWR_PVDLevelConfig(uint32_t PWR_PVDLevel);

void PWR_WakeUpPinCmd(FunctionalState NewState);//开起WAKEUP引脚的唤醒功能，不需要GPIO初始化

void PWR_EnterSTOPMode(uint32_t PWR_Regulator, uint8_t PWR_STOPEntry);//电压调节器开启或者低功耗，进入停止模式的入口参数
void PWR_EnterSTANDBYMode(void);

FlagStatus PWR_GetFlagStatus(uint32_t PWR_FLAG);
void PWR_ClearFlag(uint32_t PWR_FLAG);
```
