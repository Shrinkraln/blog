# 1. TIM输出比较OC

1. 比较CNT和CCR寄存器值的关系，来对输出电平进行置1，置0或者翻转的动作，用于输出一定频率和占空比的PWM波形
2. 每个高级定时器和通用定时器有四个输出比较通道
3. 高级定时器的前三个通道额外拥有死区生成和互补输出功能
4. 根据功能选择具体的定时器
5. PWM
   1. 占空比越大，越趋近于高电平
   2. 占空比变化的最小单位

6. 输出比较通道
   1. 通用定时器
      1. ref可以映射到主模式的控制器
      2. ref主要进入极性选择，是否翻转电平
   2. 高级定时器
   3. 输出比较模式
      1. PWM1模式下半区有效
      2. 而PWM1的不翻转极性和PWM2的翻转极性输出是一样的
      3. PWM频率和计数器更新频率一致
      4. PWM分辨率和重装载值倒数一致
      5. PWM占空比就是CRR占重装载值

# 2. 控制舵机

1.  PWM当做通信协议，相当于使能，低电平锁死，直接供电是5v stlink给直流电机
2. 直流电机：
   1. 正接正转
   2. GPIO无法直接驱动，需要单独的电机驱动电路，如TB6612
   3. TB6612双路H桥直流电机驱动芯片，可以独立驱动两个电机的转速和方向
   4. PWM控制速度，IN控制方向
   5. 同高制动，同低停止，一高一低，PWM，高转低制动
3.  在PWM频率分辨率不变的情况下，通过调节占空比，实现在确定的PWM波形中控制等效电压的大小，所以输出到的GPIO口都是AF_PP控制

# 3. 重映射复用引脚

1. 默认复用的端口，如PB4到NJTRST也需要复用端口配置gpio_pinremapconfig传入一个参数值和enbale即可，而不是指定引脚号再指定复用模式

```c
gpio_pinremapconfig(gpio_remap_swj_jtagdisable,enable);//实现下载引脚的接触重映射，恢复普通引脚
```
