# 1. FLASH

1. 程序存储器，系统储存区，选项字节；
2. 通过闪存储存器接口可以对程序存储器和选项字节进行擦除和编程
3. 读写作用
   1. 用程序存储器的剩余空间保存掉电不丢失的用户数据
   2. 通过在程序中编程IAP实现程序自我更新
4. 在线编程ICP更新程序存储器的全部内容，通过JTAG SWD BOOTLOADER
5. IAP使用微控制器支持的任一通信接口下载（自己写BOOTLOADER
6. 内存存储方式，只有PAGE 
7. 相较于ICP直接写入整个整个程序，IAP是程序在运行过程中检测到有升级指令，进入BOOT，通过接口接收新程序，写入新程序
8. 闪存容量一般只包括程序寄存器地址

```c
设备运行 → 检测到升级命令
        ↓
进入 IAP Boot
        ↓
通过通信接口接收新固件
(UART/WIFI/CAN/USB)
        ↓
擦除旧程序
        ↓
写入新程序
        ↓
跳转运行
```

15. 选项字节擦除
    1. 写使能
    2. 等待忙FLASH_SR BSY
    3. 在解锁闪存的OPTKEYR之后还有解锁FLASH_CR OPTWRE
    4. CR的OPTER1，表示即将擦除选项字节
    5. STRT1开始工作
    6. 等待BSY 0
16.  选项字节编程
    1. 写使能
    2. 等BSY
    3. 解锁OPTWRE
    4. OPTPG1，表示开始编程
    5. 写入要编程的半字到指定地址
    6. 等待BSY 0
