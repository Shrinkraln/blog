# 1. TIM定时中断

1. 功能实现：对输入的时钟进行计数，达到计数值时触发**中断**
2. 16位计数器，预分频器，自动重装寄存器的时基单元，在72MHz计数时钟下可以实现最大59.65s的定时，还可以级联嵌套
3. *不仅具备基本的定时中断功能，还包括**内外时间源选择**，输入捕获，输出比较，编码器接口，主从触发模式*
4. TIM分类：高级，通用，基本
   1. 基本定时器
      1. PSC连接内部时钟，最大65535：分频系数-1，只是调节比例
      2. 计数器最大65535，只支持向上计数模式，每来一个时钟，值+1（此时钟可以通过预分配器进行调节
      3. 自动重装寄存器：达到目标值，重装，产生UI更新中断，通往NVIC，CPU响应，或者产生U更新事件，不会产生中断，但是可以触发条件
      4. *主模式触发DAC：内部硬件不受程序控制自发运行，更新事件U映射到触发输出的位置，TRGO直接接到DAC触发引脚*
      5. 从模式触发：选择输出信号类型到CNT
   2. 通用定时器
      1. 计数器还可能向下计数和中央对齐
      2. 可以选择外部时钟
         1. TIMxETR引脚，如tim2ETR就是PA0引脚，配置触发条件等，滤波（外部输入时钟难免毛刺，采样评率越低，采样点数越多，效果越好，信号延迟越大）后：
            1. ETRF，进入触发控制器，作为时基单元的时钟（**外部时钟模式2**不会占用从模式控制器，独立于模式1
            2.  触发输入
         2. TRGI触发输入，触发**从模式**（同样是作为触发控制器的可选择信号源，**外部时钟模式1**，信号来自
            1. ETR，常用，**自带预分频器**
            2. ITR，来自其他定时器的**TRGO主模式**输出，实现**定时器的级联**
            3. TI1F_ED，**来自输入捕获单元的**ch1引脚，ed上升下降沿都有效
            4. TI1FP1，ch1引脚的时钟，仅单边有效
            5. TI2FP2，ch2引脚的时钟，仅单边有效
      3. 输出比较电路
         1. 四个通道对应ch1~4，输出PWM波形，控制电机
      4. 输入捕获电路
         1. 获取ch的方波频率
      5. 输入捕获，输出比较寄存器
         1. 两者公用，两者不能同时使用
      6. 中断信号很多，需要中断输出控制选择哪些是有效的
   3. 高级定制器
      1. 申请中断的UI多了一个重复计数器，对于输出的中断信号再次分频
      2. DTG死区生成，后输出通道两个正反，驱动三相无刷电机（三个桥臂，每个由两个大功率开发管，一共三组正反电路
      3. 死区：防止开关切换瞬间，硬件不理想造成的电磁干扰形成直通，先生成一段时间的死区，再切换下一个开关
      4. 刹车输入：BKIN产生刹车信号或者内部时钟故障，控制电路自动切断电机输出

5. 预分频器时序
   1. 计数器计数频率：CK_CNT = CK_PSC / (PSC + 1)  就是分配后的**频率**
   2. 当计数值达到重装载值且下一个时钟周期到来，计数器清零，产生中断
   3. 预分配控制寄存器：存储分频系数-1
   4. 预分频缓冲寄存器：本次计数周期结束产生更新事件，分频系数才生效
   5. 预分频计数器内部也是计数器重装载值实现

6. 计数器时序
   1. 更新中断标志位
   2. 计数器溢出频率**CK_CNT_OV = CK_CNT / (ARR + 1) = CK_PSC / ((PSC + 1) *(ARR + 1))**
   3. 产生中断的时间（计数器中断时间）=重装载值 *(1/CK_PSC) *分频系数
   4. -**CK_PSC**->预分频器-**PSC**->计数器-**重装载值**->计数器溢出->中断
   5. 设置ARPE设置是否有预装时序

7. RCC时钟树
   1. 以AHB左边为分界线，左边（不包括AHB）都是时钟的产生电路，右边是分配电路
   2. 产生电路：四个振荡源
      1. 内部8Mhz高速RC振荡器，系统时钟
      2. 外部4-16Mhz高速石英晶体振荡器，系统时钟，更稳定，PLL锁相环进行倍环，生成频率更高的72Mhz时钟
      3. 外部低速晶振，一般给RTC提供时钟
      4. 内部低速RC振荡器，一般给看门狗
      5. CSS监测外部时钟失效，跳转到内部时钟
   3. 分配电路
      1. 外设时钟使能就是赋值开启时钟
